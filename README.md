# Python Calculator

A command-line calculator built using Python.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Perform operations on multiple numbers
- Menu-driven interface
- Input validation for menu choices
- Modular design using functions

## Concepts Used

- Functions
- Dictionaries
- Lists
- Loops (`while`, `for`)
- Conditional statements
- Returning values
- Python function references

## Project Structure

- `display_menu()` – Displays the calculator menu.
- `get_choice()` – Reads the user's menu choice.
- `get_numbers()` – Reads multiple numbers from the user.
- `perform_operation()` – Executes the selected operation.
- Separate functions for each arithmetic operation.

## Example

```text
1. Add
2. Subtract
3. Multiply
4. Divide
5. Exit

Enter your choice: 1
Enter numbers separated by spaces:
5 10 15

30
```
